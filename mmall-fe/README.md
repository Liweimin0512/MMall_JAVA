## 本项目使用的工具如下
+ IDE我使用webstorm（当然使用sublime也是可以的，纯粹个人喜好）
+ 模块化方案采用CommonJS + Webpack （所以需要NodeJs）
+ 代理软件使用的是Charles（前端开发神器）
+ 架构模式：前后端完全分离，纯静态方式并且分层架构模块化
+ 版本控制工具使用git（这不是废话么...)
+ 发布方式使用编写shell脚本实现自动化发布（还是计划中）
